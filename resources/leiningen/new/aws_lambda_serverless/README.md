### Testing

You can test function locally:

```

```

You can give input (AWS events) via `stdin`.
```
lein run < event.json
```

### Serverless

AWS Lambdas are managed with [Serverless](https://serverless.com/) framework.

Install serverless dependency:
```
npm i -g serverless
```

Creating stack
```
npm install
lein uberjar
serverless deploy -v --stage dev
```
